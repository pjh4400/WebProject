# Webprogramming Project

게시판 구현 프로젝트



## 구현 기능

- 사용자 등록 / 로그인 / 로그아웃 (/)
- 소개 페이지 (/about)
- 게시판 (/board) – 읽기, 쓰기, 수정, 삭제
    - 제목, 작성자, 작성일, 본문, 해시태그
  - 사진 갤러리 (/gallery) – 읽기, 쓰기, 수정, 삭제
    - 사진, 작성자, 작성일, 해시태그



## 기술 스택

- Front-End  : `React` + `Mateiral-ui  `

- Back-End  : `Node.js` + `MonogDB`
